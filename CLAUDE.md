# Regles permanentes - Site MDPL Immigration

## Navigation et mise en page (TRES IMPORTANT)
- Toute carte, bloc ou bouton cliquable destine a montrer des articles ou des resultats DOIT mener a une page dediee (chargement complet). Jamais d'accordeon, de defilement-filtre ni d'ouverture de contenu sur place pour la navigation du blog.
- La recherche soumet vers une page de resultats (tag.html) ; elle ne filtre pas sur place.
- Les listes d'articles affichent des CARTES propres : image + titre + extrait court (1-2 phrases) + bouton "Lire l'article". Jamais le texte complet d'un article dans une liste.
- Exception : les accordeons de la FAQ (questions-frequentes.html) restent tels quels.

## Bilingue
- Site bilingue FR/EN via i18n. Toute modification doit gerer ET etre testee dans les DEUX langues.

## Code
- Guillemets ASCII droits (" et ') uniquement, jamais de guillemets typographiques.
- Ne pas casser le desktop en corrigeant le mobile (et vice-versa).
- Ne corriger que ce qui est demande ; aucune amelioration non demandee ; aucune regression ailleurs.
